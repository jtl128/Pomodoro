# Pomodoro
是一个基于番茄工作法的命令行效率工具，通过25分钟专注计时帮助用户管理时间、追踪打断原因并分析专注力数据，逐步克服拖延习惯。
# 环境配置
程序运行需要Windows操作系统，使用MinGW GCC编译时执行命令gcc fanqie.c -o pomodoro.exe -std=c99，使用Visual Studio则执行cl fanqie.c。所有依赖均为Windows标准库，无需额外安装。
# 运行步骤
编译生成可执行文件pomodoro.exe。
双击运行或在命令行中执行./pomodoro.exe。
根据菜单提示输入数字选择功能。
# 功能说明
1.开始番茄钟：启动25分钟专注计时，界面显示进度条和剩余时间。专注期间可按i键记录打断，按p键暂停计时。结束后自动进入5分钟休息，可选择继续下一个番茄钟。
2.记录打断原因：提供手机干扰、走神发呆、环境干扰、聊天和其他原因五个选项，自动记录发生时间和所属番茄钟序号。
3.生成专注力报告：展示完成番茄钟数、专注总时长和打断次数，统计打断原因分类，并根据打断比例计算专注力评分，给出相应建议。
4.获取今日动力：随机显示一条励志语录，在需要鼓励时提供精神支持。
5.数据管理：所有数据自动保存到程序目录下的pomodoro_data.txt文件，包含每日统计和打断记录。程序启动时会加载最近一条历史记录。
